This repository contains all the course code to learn how to use Hygraph (Headless CMS) in React with Next.js.

The project demonstrates the integration of a headless CMS using GraphQL to manage and display dynamic content, along with modern styling using Tailwind CSS.

### Prerequisites

## ðŸ”§ Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/abrahamgalue/hygraph-with-next-course.git
   cd hygraph-with-next-course
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

### Local Execution

Start the development server (enabled with Turbopack):

```bash

# or
```
